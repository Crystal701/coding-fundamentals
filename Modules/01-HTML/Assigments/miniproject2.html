HTML Block Elements

This paragraph and the h1 element right above are block elements, and so is the section element wrapping around this section.
Some other block elements include:

  • ul, ol, li
  • nav, header, main, footer
  • table
  • form
  • article

HTML Inline Elements

The elements used to bold and italize these texts are inline elements, and so are the span and anchor link wrapped around here.
